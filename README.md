# react-codemod-patched

Patched local version to support recent Flow syntaxes.   
See https://github.com/reactjs/react-codemod/issues/314

## Install

```
$ cd react-codemod-patched
$ npm i
$ npm link
```

## Usage

```
$ react-codemod-patched
```

Or for instance

```
react-codemod-patched update-react-imports
```
